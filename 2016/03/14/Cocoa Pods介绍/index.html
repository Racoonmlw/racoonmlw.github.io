Cocoa Pods介绍

Cocoa Pods是一个负责管理iOS项目中第三方资源的工具，只需要将用到的第三方开源库放到一个名为Podfile的文件中，然后在命令行执行pod install命令，CocoaPods就会自动将这些第三方资源库的源代码下载下来，并且为当前的工程设置好响应的系统依赖和编译参数。

安装：

  1. 如何在MAC系统下安装Pod工具
    － 安装的方式非常简单，Mac下已经再带了ruby，只要使用ruby的gem命令就可以安装了
    － 打开Mac终端工具，在终端运行下面的命令，sudo gem install cocoapods pod setup
    －如果失败，运行下面命令
    gem sources —remove https://rubygems.org/
    gem sources -a http://ruby.taobao.org/
    gem sources -l
    sudo gem install cocoapods
    － 如何修复404错误，在终端运行如下命令：
    gem sources –remove http://ruby.taobao.org/
    gem sources -a https://ruby.taobao.org/
  2. 更新相关步骤
    － 如果我们修改了Podfile文件，那么就需要更新CocoaPods，使用下面的ruby的gem命令来更新，sudo gem update cocoapods
  3. 如何使用pod查找第三方库
    －可以使用pod search + lib_name来搜索第三方库，查看版本，如下命令所示，sudo pod search SDWebImage

如何在Xcode环境下使用Cocoa Pods工具

  1. 步骤 (下面涉及到的命令需要在Mac终端运行)
    － 第一步: 使用cd命令切换到你的项目所在的路径,例如：cd /Users/xiao/Documents/Demo-Pods
    － 第二步: 接着在终端使用 pod init命令创建Podfile，执行如下命令，可以看到项目下面创建了一个Podfile文件,例如：pod init
    － 第三步: 打开Podfile文件夹 添加 pod ‘MJRefresh’ 这样的形式的代码，表示要引入这个框架 (一般可以依照github网站上的README.md
    中要求的填写)
    － 第四步: 在终端输入命令行 pod update 更新Podfile中的所有第三方库
    － 备注：
    a. 如何打开Xcode项目 使用cocoapod⽅方式，更新完成后会自动添加workspace文件，以后工程都要双击以workspace后缀结尾的文件
    b. 引入pod方式加载的框架头文件 ，需要使用#import <>
